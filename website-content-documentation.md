## Design System Notes

### Color Palette
- Primary brand color: #A1BCD1 (light blue)
- Accent highlights: #A8CDFF (bright blue)
- Success/positive: Green variations
- Warning/negative: Red variations

### Typography
- Primary font: Geist Sans
- Hierarchy: 6xl/7xl for main headlines, 4xl for section headings, xl-3xl for subsections

### Layout Patterns
- Full-width hero sections
- Centered content with max-width containers
- Grid layouts: 2-column for comparisons, 3x2 for problem cards, Bento grid for features
- Scroll-triggered animations
- Light/dark mode support throughout

### Interactive Elements
- Scroll-animated step progression
- Theme toggle
- Hover effects on cards and buttons
- Animated text rotation in hero
- Sticky positioning for step visuals

### Content Strategy
- "Your workforce" language pattern for personalization
- Direct address to reader ("You pay for...", "Your Digital Workforce...")
- Outcome-focused messaging
- Strategic executive language
- Technical capability descriptions balanced with business benefits
