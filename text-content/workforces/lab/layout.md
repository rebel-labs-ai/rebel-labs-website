# Workforce Lab Page Layout Documentation

## Page Structure Overview

The Lab page follows a vertical flow design emphasizing custom workforce development and the partnership between Novosapien and Rebel Labs.

### 1. Navigation (Fixed Header)

- **Component**: Navigation component
- **Position**: Fixed top navigation with backdrop blur
- **Content**: Logo + navigation links + CTA button
- **Interactive Elements**: Dropdown for "Workforces" with hover states

### 2. Hero Section

- **Background**: Subtle gradient with animated dots pattern
- **Layout**: Centered card with backdrop blur effect
- **Text Elements**:
  - Badge text (~3 words)
  - Main headline (~4 words)
  - Description paragraph (~25 words)
  - CTA button
- **Visual Elements**:
  - Card with border accent
  - Shadow effects for depth
  - Responsive padding

### 3. Division of Labor Section

- **Layout**: Two-part section with partnership and collaboration blocks
- **Background**: Accent color at 40% opacity
- **Text Elements**:
  - Section headline (~5 words)
  - Section description (~35 words)
  - Partnership descriptions (2 blocks, ~25 words each)
  - Extended philosophy text (~120 words)
- **Visual Elements**:
  - Logo lockup (Novosapien + Rebel Labs)
  - Two-column card grid for Team vs Workforce
  - Icon elements (Brain, Layers)
  - Hover effects on cards

### 4. Engagement Protocol Section (Interactive)

- **Layout**: Timeline-based process visualization
- **Text Elements**:
  - Section headline (~6 words)
  - Timeline indicator (~6 words)
  - 4 process steps with badges, headlines (~3 words each), and descriptions (~45 words each)
- **Interactive Elements**:
  - Animated timeline progression
  - Step-by-step reveal animations
  - Visual progress indicators

### 5. Art of the Possible Section

- **Layout**: 2x2 grid of possibility cards
- **Background**: Gradient overlay (accent/10)
- **Text Elements**:
  - Section headline (~5 words)
  - Introduction text (~10 words)
  - Key question (~4 words)
  - 4 possibility cards (~30 words each)
- **Visual Elements**:
  - Icon for each card (Search, TrendingUp, Target, BarChart3)
  - Card hover effects
  - Bold text emphasis within descriptions

### 6. Final CTA Section

- **Layout**: Centered card design
- **Text Elements**:
  - Badge (~5 words)
  - Headline (~3 words)
  - Description paragraph (~30 words)
  - CTA button
- **Visual Elements**:
  - Card with backdrop blur
  - Accent border styling

### 7. FAQ Section

- **Layout**: Accordion-style expandable items
- **Text Elements**:
  - 5 questions (~8-12 words each)
  - 5 detailed answers (~40-60 words each)
- **Interactive Elements**:
  - Click to expand/collapse
  - Smooth height animations
  - Plus/minus indicators

### 8. Related Solutions Section

- **Layout**: Two-column card grid
- **Text Elements**:
  - Section headline (~4 words)
  - Section description (~15 words)
  - 2 workforce cards with names and descriptions (~20 words each)
- **Visual Elements**:
  - Card borders and shadows
  - Arrow indicators for links

### 9. Footer

- **Layout**: Four-column grid with bottom section
- **Text Elements**: Standard footer with navigation and company information

## Interactive Elements

1. **Navigation Dropdown**: Hover-triggered workforce options
2. **Card Hover Effects**: Border and shadow transitions
3. **Timeline Animation**: Step-by-step process visualization
4. **FAQ Accordions**: Expandable question/answer panels
5. **Button Hover States**: Multiple CTA buttons with state changes
6. **Logo Partnership Display**: Visual brand lockup
7. **Icon Animations**: Subtle icon effects on hover

## Responsive Behavior

- **Desktop**: Full grid layouts, side-by-side cards
- **Tablet**: Adjusted column counts, maintained hierarchy
- **Mobile**: Single column stack, adjusted typography sizes
  - Partnership logos stack vertically on mobile
  - Process steps become vertical timeline
  - Possibility cards stack in single column

## Visual Design Elements

### Color Usage:
- **Primary Accent**: Used for badges, borders, and emphasis
- **Card Backgrounds**: Semi-transparent with backdrop blur
- **Hover States**: Increased border opacity and shadow depth
- **Text Hierarchy**: Foreground for headlines, muted for descriptions

### Typography:
- **Headlines**: text-2xl to text-4xl (responsive scaling)
- **Body Text**: text-base to text-lg
- **Badges**: text-xs to text-sm with uppercase styling
- **Emphasis**: Bold text within descriptions for key points

### Spacing:
- **Section Padding**: py-16 to py-32 (responsive)
- **Card Padding**: p-6 to p-12 (responsive)
- **Content Max Width**: max-w-6xl for most sections
- **Grid Gaps**: gap-4 to gap-8 (responsive)

## Content Guidelines

- Headlines use clear, action-oriented language
- Descriptions balance technical depth with accessibility
- Partnership messaging emphasizes collaboration
- Process steps focus on tangible outcomes
- FAQ provides transparent, detailed information
- CTAs use direct, compelling language
